# Form Align Website

Marketing website for the Form Align smart Pilates mat, built as a portfolio-ready landing page with:

- Product overview
- Technology highlights
- Dedicated `Design Process` section (engineering workflow summary)
- Outcomes and contact section

## Run locally

Open `index.html` in any browser.

## Deploy with GitHub Pages

This project is static and can be deployed from the `main` branch root:

1. Push this repository to GitHub.
2. In GitHub: **Settings -> Pages**.
3. Set **Source** to `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save and wait for the Pages URL to be published.

## Customize

- Brand name and text: `index.html`
- Theme and layout: `styles.css`
- Mobile menu behavior: `script.js`
